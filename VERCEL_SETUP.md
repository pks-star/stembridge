# Vercel Environment Variables Setup

## Add these in Vercel Dashboard:

1. Go to: https://vercel.com/pks-star/stembridge/settings/environment-variables

2. Add each variable:

| Key | Value |
|-----|-------|
| `NEXT_PUBLIC_SUPABASE_URL` | Your Supabase project URL |
| `NEXT_PUBLIC_SUPABASE_ANON_KEY` | Your Supabase anon key |

3. Click **Save** and **Redeploy**

---

## That's it! 🎉

Your app will be live at: **https://stembridge-psi.vercel.app**

---

**Note:** The AI chat feature requires Ollama running locally. For production AI, you'll need to connect to a cloud AI service later.
